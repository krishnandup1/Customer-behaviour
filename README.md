# Customer Shopping Behavior Analysis

## Project Overview

This project analyzes customer shopping behavior using Python, SQL, PostgreSQL, and Power BI to uncover purchasing patterns, customer segments, and business insights. The objective is to transform raw retail data into actionable insights that can support strategic decision-making, customer retention, and revenue growth.

---

## Business Problem

Retail businesses generate large volumes of customer transaction data. Without proper analysis, it becomes difficult to identify:

* High-value customers
* Purchasing trends
* Customer loyalty patterns
* Revenue-driving segments
* Opportunities for targeted marketing campaigns

This project addresses these challenges by building an end-to-end analytics workflow from data preparation to dashboard visualization.

---

## Objectives

* Analyze customer demographics and purchasing behavior
* Identify key customer segments
* Evaluate spending patterns and loyalty metrics
* Generate business insights through SQL analysis
* Build interactive dashboards for decision-making

## Project Workflow

### 1. Data Collection

* Imported customer shopping dataset.
* Reviewed data structure and business attributes.

### 2. Data Cleaning & Preprocessing

* Handled missing values.
* Removed duplicates.
* Standardized data formats.
* Performed data validation checks.

### 3. Exploratory Data Analysis (EDA)

* Customer demographic analysis
* Purchase frequency analysis
* Spending behavior analysis
* Category-wise sales analysis
* Loyalty trend analysis

### 4. Database Design

* Created relational tables in PostgreSQL.
* Loaded cleaned datasets into SQL database.
* Established relationships between tables.

### 5. SQL Analysis

Performed analytical queries using:

* Joins
* Aggregations
* Common Table Expressions (CTEs)
* Window Functions
* Ranking Functions
* Customer Segmentation Analysis

### 6. Dashboard Development

Built interactive Power BI dashboards featuring:

* Customer KPIs
* Spending Trends
* Revenue Analysis
* Customer Segmentation
* Purchase Distribution
* Loyalty Metrics
